# API Layer (FastAPI) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development
> (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use
> checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wrap the finished upload-catalog backend (pure functions taking a DB `conn`) in a FastAPI
HTTP layer implementing the endpoint table of `docs/superpowers/specs/2026-07-05-frontend-api-handoff.md`.

**Architecture:** One FastAPI app built by a `create_app()` factory. Each request gets its own
psycopg connection/transaction via a `get_conn` dependency (commit on success, rollback on error).
A stub-auth dependency (`get_identity`) resolves the subject + roles from `X-User`/`X-Roles` headers
— it is the single M6 seam to swap for a real IdP later. Endpoints are thin: decode inputs, call the
backend function, return its dataclasses (FastAPI serializes stdlib dataclasses natively).
LLM feature-assist is gated behind a fail-closed 503 until the real-provider follow-on.

**Tech Stack:** Python 3.11, FastAPI + Pydantic v2, psycopg 3 (sync), uvicorn, pytest +
`fastapi.testclient` (httpx), uv for all commands.

## Global Constraints

- Python `>=3.11`; ruff line-length 100; `make ci` (ruff + format-check + mypy + pytest) must pass.
- Run everything through uv: `uv run pytest …`, `uv run ruff check …`, `uv run mypy`.
- **M6 read-scope:** `roles` reach `search`/`recommend_features`/`feature_recipe` ONLY from
  `get_identity` (the session seam). Never accept roles via query/body params.
- **First-class states, not errors:** `ingested|held|rejected` (+ `flagged`, `quarantined` counts)
  are HTTP 200 response states. Only transport/validation problems are 4xx.
- **Enrichment honesty:** `ingest_upload(..., client=None)` always (no LLM enrichment at ingest);
  feature-assist endpoints return 503 unless a client was injected (tests inject a scripted FakeLLM).
- Endpoints are sync `def` (psycopg is sync; FastAPI runs them in its threadpool).
- Backend functions are the source of truth — wrap them, never reimplement or "improve" them.
- Endpoints return the backend dataclasses directly (FastAPI serializes stdlib dataclasses). If
  response-model generation ever fights a frozen/slots dataclass, set `response_model=None` on that
  route and keep returning the dataclass — never hand-copy fields into a parallel model.
- Tests live in `tests/featuregen/api/` so they inherit the repo fixtures: root `conn` (rolled back
  per test, migrations applied once) and `tests/featuregen/conftest.py` (phase-06 event schemas,
  fake identity verifier, `db` alias).
- Commit messages: conventional (`feat(api): …`, `docs: …`), ending with
  `Co-Authored-By: Claude Fable 5 <noreply@anthropic.com>`.

from fastapi.testclient import TestClient

                yield conn

            app.dependency_overrides[get_conn] = _test_conn
            return stack.enter_context(TestClient(app))

        yield _make


@pytest.fixture
def client(make_client):
    return make_client()

`src/featuregen/api/deps.py`:

```python
"""Request-scoped dependencies: DB transaction, stub session auth, fail-closed LLM gate."""
from __future__ import annotations

from collections.abc import Iterator

import psycopg
from fastapi import Header, HTTPException, Request

from featuregen.config import get_settings
from featuregen.contracts.envelopes import IdentityEnvelope
from featuregen.intake.llm import LLMClient


def get_conn() -> Iterator[psycopg.Connection]:
    """One connection + transaction per request: commit on success, rollback on any error.
    Ingest stays all-or-nothing; reads never leave dangling transactions. Tests override this."""
    dsn = get_settings().dsn
    if not dsn:
        raise HTTPException(status_code=503, detail="FEATUREGEN_DSN is not configured")
    conn = psycopg.connect(dsn)
    try:
        yield conn
        conn.commit()
    except Exception:
        conn.rollback()
        raise
    finally:
        conn.close()


def get_identity(
    x_user: str | None = Header(default=None),
    x_roles: str = Header(default=""),
) -> IdentityEnvelope:
    """Stub session auth (spec build-step 1): subject + roles from headers until the real IdP
    lands. This dependency is the M6 seam — swap it for real session resolution without touching
    any endpoint. Roles must never be accepted from request params or bodies."""
    if not x_user:
        raise HTTPException(status_code=401, detail="missing X-User header (stub auth)")
    roles = tuple(r.strip() for r in x_roles.split(",") if r.strip())
    return IdentityEnvelope(subject=x_user, actor_kind="human", authenticated=True,
                            auth_method="stub", role_claims=roles)


def get_llm(request: Request) -> LLMClient:
    """Fail closed: feature-assist needs an injected LLM client (tests/dev pass a scripted
    FakeLLM). None in production until the real-provider follow-on — 503, not a broken fake."""
    client = getattr(request.app.state, "llm_client", None)
    if client is None:
        raise HTTPException(
            status_code=503,
            detail="feature-assist LLM is not configured on this deployment "
                   "(real-provider enrichment follow-on pending)")
    return client
```

"""FastAPI app factory. Run: uvicorn --factory featuregen.api.app:create_app"""
from __future__ import annotations

from collections.abc import AsyncIterator
from contextlib import asynccontextmanager

from fastapi import FastAPI

from featuregen.aggregates.bootstrap import register_phase06_event_schemas
from featuregen.intake.llm import LLMClient
from featuregen.overlay.config import overlay_config_from_env, register_overlay_config


@asynccontextmanager
async def _lifespan(app: FastAPI) -> AsyncIterator[None]:
    # The same process bootstrap the worker and the test suite use: event schemas (idempotent)
    # + the sealed overlay config (fail-closed accessor needs it registered before any ingest).
    register_phase06_event_schemas()
    register_overlay_config(overlay_config_from_env())
    yield


def create_app(llm_client: LLMClient | None = None) -> FastAPI:
    app = FastAPI(title="FeatureGen API", lifespan=_lifespan)
    app.state.llm_client = llm_client

    @app.get("/health")
    def health() -> dict[str, str]:
        return {"status": "ok"}

    return app
```

from __future__ import annotations

from datetime import datetime, timezone

import psycopg
from fastapi import APIRouter, Depends, File, Form, HTTPException, UploadFile

from featuregen.api.deps import get_conn, get_identity
from featuregen.contracts.envelopes import IdentityEnvelope
from featuregen.overlay.upload.canonical import CanonicalRow
from featuregen.overlay.upload.csv_reader import read_csv_rows
from featuregen.overlay.upload.excel_reader import read_excel_rows
from featuregen.overlay.upload.ingest import IngestResult, ingest_upload

router = APIRouter()


def _read_rows(filename: str, data: bytes, source: str) -> list[CanonicalRow]:
    name = filename.lower()
    if name.endswith((".xlsx", ".xlsm")):
        return read_excel_rows(data, source=source)
    if name.endswith(".csv"):
        return read_csv_rows(data.decode("utf-8-sig"), source=source)
    raise HTTPException(status_code=400, detail="unsupported file type (expected .csv or .xlsx)")


@router.post("/uploads")
def create_upload(
    file: UploadFile = File(...),
    source: str = Form(...),
    conn: psycopg.Connection = Depends(get_conn),
    identity: IdentityEnvelope = Depends(get_identity),
) -> IngestResult:
    try:
        rows = _read_rows(file.filename or "", file.file.read(), source)
    except HTTPException:
        raise
    except Exception as exc:   # a malformed file is a client error, not a 500
        raise HTTPException(status_code=400, detail=f"could not parse upload: {exc}") from exc
    # client=None: no LLM enrichment at ingest until the real-provider follow-on (spec guardrail).
    return ingest_upload(conn, source, rows, actor=identity,
                         now=datetime.now(timezone.utc), client=None)
```

from __future__ import annotations

from datetime import datetime, timezone

import psycopg
from fastapi import APIRouter, Depends, Query

from featuregen.api.deps import get_conn, get_identity
from featuregen.contracts.envelopes import IdentityEnvelope
from featuregen.overlay.upload.search import SearchHit, search

router = APIRouter()


@router.get("/search")
def search_catalog(
    q: str,
    limit: int = Query(default=20, ge=1, le=100),
    conn: psycopg.Connection = Depends(get_conn),
    identity: IdentityEnvelope = Depends(get_identity),
) -> list[SearchHit]:
    # Roles come from the authenticated session — NEVER from the request (M6 read-scope).
    # Freshness fail-closed is enforced inside search(): a stale source's rows are absent.
    return search(conn, q, now=datetime.now(timezone.utc),

from __future__ import annotations

from datetime import datetime, timezone

import psycopg
from fastapi import APIRouter, Depends, HTTPException
from pydantic import BaseModel, Field

from featuregen.api.deps import get_conn, get_identity
from featuregen.contracts.envelopes import IdentityEnvelope
from featuregen.overlay.upload.features import (
    FeatureFreshness, FeatureSpec, feature_freshness, features_affected_by, register_feature)

router = APIRouter()


class DerivesFromIn(BaseModel):
    catalog_source: str
    object_ref: str


class FeatureSpecIn(BaseModel):
    name: str = Field(min_length=1)
    description: str = ""
    grain_table: str | None = None
    aggregation: str | None = None
    as_of_column: str | None = None
    derives_from: list[DerivesFromIn] = []


@router.post("/features")
def create_feature(
    body: FeatureSpecIn,
    conn: psycopg.Connection = Depends(get_conn),
    identity: IdentityEnvelope = Depends(get_identity),
) -> dict[str, str]:
    """Registration is the explicit-confirm step — suggestions only become features here."""
    spec = FeatureSpec(
        name=body.name, description=body.description, grain_table=body.grain_table,
        aggregation=body.aggregation, as_of_column=body.as_of_column,
        derives_from=tuple((d.catalog_source, d.object_ref) for d in body.derives_from))
    return {"feature_id": register_feature(conn, spec)}


@router.get("/features/{feature_id}/freshness")
def freshness(
    feature_id: str,
    conn: psycopg.Connection = Depends(get_conn),
    identity: IdentityEnvelope = Depends(get_identity),
) -> FeatureFreshness:
    row = conn.execute("SELECT 1 FROM feature WHERE feature_id = %s", (feature_id,)).fetchone()
    if row is None:
        raise HTTPException(status_code=404, detail=f"unknown feature {feature_id!r}")
    return feature_freshness(conn, feature_id, now=datetime.now(timezone.utc))


@router.get("/columns/{object_ref}/feature-impact")
def feature_impact(
    object_ref: str,
    source: str,
    conn: psycopg.Connection = Depends(get_conn),
    identity: IdentityEnvelope = Depends(get_identity),
) -> dict[str, list[str]]:
    return {"feature_ids": features_affected_by(conn, source, object_ref)}
